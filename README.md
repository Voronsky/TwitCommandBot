# TwitCommandBot
Send commands to your machines via twitter, because why not.

# How to Use
```shell
   twitCommandBot -f credentials.txt
```

# Dependencies
  There are a list of modules that need to be installed due to certain SMTP services providing their own API
  * Sparkpost-python
  * Tweepy
  * requests
  * smtplib
